Supabase 接続設定

    
SupabaseプロジェクトのURLとAnon Keyを入力してください。
情報はブラウザに保存されます（このデバイスのみ）。

    

    

    
⚠️ anon（公開）キーを使用してください。
SupabaseダッシュボードのProject Settings → API → anon publicに記載。
Row Level Security（RLS）で管理者のみアクセス可能に設定することを推奨します。
